## 🔧 MANUÁLNA INŠTALÁCIA (ak je potrebné):

### Linux:
```bash
# Ikonka
mkdir -p ~/.local/share/icons/hicolor/256x256/apps
cp naklady-icon.svg ~/.local/share/icons/hicolor/256x256/apps/

# Desktop
mkdir -p ~/.local/share/applications
cp naklady.desktop ~/.local/share/applications/

# Aktualizácia cache
update-desktop-database ~/.local/share/applications/
```

### Windows:
```cmd
REM Skopíruj naklady.cmd na plochu
copy naklady.cmd "%USERPROFILE%\Desktop\naklady.lnk"
```
