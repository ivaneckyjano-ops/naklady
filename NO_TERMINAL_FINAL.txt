## ✨ ČO BOLO VYKONANÉ:

### 1. Vytvorený nový script: `naklady-app`
- Spúšťa aplikáciu **bez viditeľného terminálu**
- Automaticky otvára prehliadač na `http://localhost:5004`
- Čaká kým je server dostupný
- Všetko beží na pozadí

### 2. Aktualizovaný `naklady.desktop`
```
STARÝ:    Terminal=true
NOVÝ:     Terminal=false
```
- Už sa neotvára žiaden terminál!

### 3. Aktualizovaný alias `naklady`
```
STARÝ:    alias naklady='/home/narbon/Aplikácie/expense-tracker/start.sh'
NOVÝ:     alias naklady='/home/narbon/Aplikácie/expense-tracker/naklady-app'
```

## 🔧 TECHNICKÉ DETAILY:

### naklady-app Script:
```bash
#!/bin/bash
# 1. Preverí virtuálne prostredie (vytvorí ak treba)
# 2. Zatvri starý proces (pokiaľ beží)
# 3. Spustí Flask na pozadí: nohup python run.py &
# 4. Čaká kým server odpovedá na localhost:5004
# 5. Otvorí prehliadač: xdg-open, open, firefox, chrome
# 6. Výstup do: /tmp/naklady.log
```

### Desktop Konfigurácia:
```ini
[Desktop Entry]
Terminal=false          ← ZMENA: Bolo true
Exec=...naklady-app     ← ZMENA: Bolo start.sh
```

## 🐛 RIEŠENIE PROBLÉMOV:

### Aplikácia sa nespúšťa z ikonky

**Riešenie:**
```bash
# Skontroluj logy
tail -f /tmp/naklady.log

# Spusti manuálne
/home/narbon/Aplikácie/expense-tracker/naklady-app
```

### Prehliadač sa neotvorí automaticky

**Riešenie:**
```bash
# Nainštaluj xdg-utils
sudo apt-get install xdg-utils

# Alebo otvori ručne: http://localhost:5004
```

### Port 5004 je obsadený

**Riešenie:**
```bash
# Zatvri starý proces
pkill -f "python.*run.py"

# Alebo zmení port v run.py
```
