# 💰 Sledovanie Domácich Nákladov

Toto je kanonická verzia aplikácie. Ak niekde narazíš na starú kópiu `expense-tracker`, používaj tento priečinok: `naklady-only`.

## Spustenie

```bash
cd /home/narbon/naklady-only
chmod +x start.sh
./start.sh
```

Potom otvor:

```text
http://localhost:5004
```

## Čo aplikácia podporuje

- výdavky podľa kategórií a podkategórií
- oblasti ako `Nevyhnutné potreby`, `Chcenia`, `Investície`, `Rezerva`
- príjmy
- štatistiku podľa kategórií aj oblastí
- porovnanie rokov
- rozbaľovanie kategórií v zozname

## Dôležité

Táto verzia je jediná, ktorú máš ďalej používať pre vývoj aj spúšťanie. Starší priečinok `expense-tracker` je len zvyšková kópia.
